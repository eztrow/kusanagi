# KUSANAGI
KUSANAGI - 34 key, wireless (handwired) keyboard.

![kusanagi](images/banner.jpg)

## Features

* 34 key
* Kailh Choc V1 switches
* Choc spacing
* Hotswap Sockets
* Wireless
* Power switch and reset button
* Status LEDs
